# 💫 About Me
Saya adalah individu yang memiliki minat besar di bidang **Teknologi Informasi**, dengan fokus pada **Network Engineering, Cyber Security, dan Software Development**. Saya memiliki pengalaman dalam membangun dan mengelola project berbasis web, jaringan, serta pengembangan aplikasi menggunakan **Python** dan **JavaScript**.

Saat ini, saya aktif memperdalam kemampuan dalam **penetration testing, web security, dan system administration (Linux & networking)**. Saya terus mengembangkan skill melalui praktik langsung seperti **Capture The Flag (CTF)**, eksplorasi tools security, serta pengembangan project berbasis **Artificial Intelligence**.

Saya dikenal sebagai pribadi yang **cepat belajar, problem solver, dan adaptif terhadap teknologi baru**. Saya memiliki semangat tinggi untuk berkembang menjadi profesional IT yang mampu memberikan solusi **efektif, aman, dan inovatif**, khususnya dalam **keamanan sistem dan infrastruktur jaringan**.
